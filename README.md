# 蓝承工业设备官方网站

这是一个展示蓝承工业设备产品的响应式网站，包含泵、止回阀和真空泵等产品信息。

## 项目概述

该网站展示了蓝承工业设备公司的产品线，包括：
- 各种型号的工业泵
- 止回阀产品
- 真空泵设备

## 部署到 Vercel

点击下方按钮可一键部署到 Vercel：

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/lancheng-website)

或者按照以下步骤手动部署：

1. 将此代码库推送到 GitHub/GitLab/Bitbucket
2. 登录 [Vercel](https://vercel.com/)
3. 点击 ""New Project""
4. 导入您的仓库
5. 点击 ""Deploy""

## 本地开发

要本地预览网站，您可以：

1. 使用 Live Server 扩展（VS Code）
2. 使用 Python 内置服务器：
   `ash
   python -m http.server 8000
   `
3. 使用 Node.js 的 http-server：
   `ash
   npx http-server
   `
